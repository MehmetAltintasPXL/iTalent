Easi: Microsoft Cloud - Azure & Intune

Inleidende sessie over de fundamenten en toepassingen van Microsoft Azure en Intune.

PXL, G410 – 12 maart 2024, 08:30 - 11:00

Deze inleidende sessie werd verzorgd door Easi en behandelde de fundamenten en toepassingen van Microsoft Azure en Intune. Tijdens de seminarie werd uitgelegd hoe organisaties cloudoplossingen kunnen inzetten voor meer flexibiliteit, schaalbaarheid en beveiliging. Er werd ook aandacht besteed aan het beheer van apparaten en gebruikers via Intune.

Wat heb ik geleerd?

  • De basisprincipes van Microsoft Azure en de voordelen van cloud computing.
  • Hoe Intune wordt gebruikt voor device management en security policies.
  • Praktische voorbeelden van implementatie in een bedrijfscontext.
